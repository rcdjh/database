2º conteudo
Tendo o projeto a necessidade de acesso em varias base de dados, qual o principal desafio de um projeto?
	Quando temos uma ideia do que vamos trabalhar com a utilização do BI, antes de tudo temos que ter transformados os dados transacionais para dados analiticios e seguindo o processo de colocar os dados em uma linha do tempo, de acordo com o processo que a empresa esta estimando encontrar o seu devido resultado.
	Os dados transacionais esta em varias bases diferente, com um projeto de BI, podemos mesclar as informações entre as demais informações, a ideia é unir todas as base, deixando elas mais claras, com informações que não foge da linha do tempo em que estamos trabalhando, se temos uma dado em uma base de dado e ela tem relação com outra base de dados, o ETL tem como finalidade preservar esse conteudo para que nada se perca ao longo do tratamento desses dados.
	É um desafio que normalmente acontece diariamente na vida de um profissional de BI, juntar as informações em varias bases de dados é um cenario que é normal no dia a dia dos profissionais de BI
